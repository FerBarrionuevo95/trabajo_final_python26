Nombre: Fernanda Barrionuevo
Comisión: 96125
¡Bienvenido a mi proyecto final! Este blog ha sido desarrollado como parte de la **Diplomatura en Python**. Es una plataforma funcional que permite a los usuarios compartir experiencias viajeras, registrarse y gestionar sus perfiles.

## 🚀 Funcionalidades
* **Gestión de Usuarios**: Registro, inicio de sesión y cierre de sesión.
* **Perfiles Personalizados**: Cada usuario puede editar su información y cambiar su contraseña.
* **Blog con CKEditor**: Los posts soportan texto enriquecido y visualización detallada.
* **Diseño Responsive**: Interfaz moderna construida con **Bootstrap 5**, adaptable a dispositivos móviles.
* **Sección Acerca de mí**: Información sobre el desarrollador con fotografía personal.

## 🛠️ Tecnologías Utilizadas
* **Backend**: Django
* **Base de Datos**: SQLite3
* **Frontend**: HTML5, CSS3, Bootstrap 5
* **Editor de Texto**: Django-CKEditor
* **Control de Versiones**: Git & GitHub
